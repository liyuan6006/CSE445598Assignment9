Michael Syrjala Global.asax: Active Session Counter
 Uses a thread safe collection to keep track of the number of active sessions that the application currently has. This information is then provided to the user in the UserControl. Given the anticipated user load it is not anticipated that any performance issues should result.
 Utilizes a thread safe ConcurrentStack bound to an Application state variable. The number of items in the stack is the count of active sessions.
